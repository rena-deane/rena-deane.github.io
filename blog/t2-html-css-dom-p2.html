Grid Based Design

The grid based design is a CSS framework used to arrange web content into rows and columns. Every webpage is based on some type of grid design. The most common grid size is 12 columns.

Responsive Webpages

People access the web using devices that vary in size/resolution. Responsive design allows a page to respond to the changing size of a web browser meaning that if the browser width is reduced the content responds so that it fits within the browser's boundaries.

Semantic Structure

Semantic tags are used so that users such as developers and the web can easily determine the infromation structure of a webpage. For example, h1 to h6 for headings, img for images and forms for forms. These tags aren't about visual styling. For instance, screen readers/assistive technology focus on semantic structures to read the page to a visually impaired user.

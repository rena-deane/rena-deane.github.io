Data-Types

  • Numbers - written with and without decimals. They can also be written with scientific (exponential) notation.
  • String - strings are made up of various characters, for example "My-String 2". They are written in quotes marks. Double or single.
  • Boolean - they can only have two values; true or false.
  • Object - JavaScript objects are written with curly braces. Object properties are written as name:value pairs, separated by commas. 
    var person = {
    firstName:"Rena", 
    lastName:"Deane",
    age:21, 
    eyeColor:"brown"};
    To access the values of the object properties we can use bracket notation or dot notation. var name = person.firstName + " " + person.firstName will return "Rena Deane"

Array - what and how?

Array - these are written within square brackets []. Array items are seperated by commas and can include a mix of data types. For example, ["Text", 54, true, "Test Array"]. Using the example array we will explore array's zero-based indexes, which means the first item is [0] = "Text", second is [1] = 54, and so on.

For more information on arrays visit JavaScript Arrays.

SCOPE in JavaScript

Scope in JavaScript is the set of variables, functions and objects we have access to.

Local Scope

Variables declared within a function become LOCAL to the function. They can only be accessed within the variable as they have local scope.

Global Scope

Variables declared outside a function are GLOBAL. They can be accessed by all scripts and functions on a webpage.

  • Automatically Global: If you assign a value to a variable that hasn't been declared then it automatically becomes Global. Do not create a glocal variable unless you intend to.

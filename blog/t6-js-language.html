JavaScript Language

parentheses ()
  • Used to group expressions. For example (1+2).
  • A function is a block of code designed to perform a particular task. When you define a function you name it and add parameters to the parentheses function myFunction(parameter1,parameter2). To call a function you add parameter values to the parentheses and these are used to execute the function.
  • They are used to surround conditional (if, else, do, while, for) statements.
brackets []
  • These are used to define arrays, list of data. For example, var array = ["words", 4, true];
  • Used to access object properties using index numbers or iterators.
braces {}
  • {} are used to create objects. {} within {} are objects within object.
  • Curly braces are also used to encapsulate pieces of code that needs to be executed as one.
single quotes ' '

Single quotes can be used to create a string - text values. However, when you create a string using single quotes and the text contains apostrophes you must add / before it. Otherwise it closes the quote up to the apostrophe.

double quotes " "
  • Double quotes are used to create a string - text values. Unlike single quotes, apostrophe's do not need to be escaped. However, if you would like to include double quotes to a string then you must escape them using /.
  • Double quotes can be used to define an object property when using literal or bracket notation.

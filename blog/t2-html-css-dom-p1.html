About HTML, CSS & DOM

The easiest way is to use the analogy used in the Udacity lesson. It's like building a house.

  • html defines the structure of the house. Which room goes where, where do the walls go, the lounge area and so on and so forth.
  • CSS styles the house. Colour of the walls, will there be carpet or floor boards.
  • I interpret DOM to be the project manager of the build. They interact with the structure and styles to build the house. They define the logical structure of how it should be built.
Boxifying and the Box Model

Boxifying explains how every element on a web page can be represented by a box. Front-end developers take a visual design and box the visual elements write down until they reach a simple element such as an image or paragraph. The boxes illustrate how elements are arranged on a page.

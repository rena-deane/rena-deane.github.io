JavaScript Basics

JavaScript compare to HTML and CSS

All three components have different jobs to do.

HTML structures content on a webpage using various elements. CSS styles the page by telling the browser how to display each element. It can vary for different media types and device sizes.

JavaScript (JS) extends the functionality of the website and allows for real time interactions. We utilise JS for visual effects through to processing and calculating data. One of my favourite functions through JS is the hamburger icon and it has become very common at collapsing the navigation menu. Usually seen when the browser is reduced in size it can also be found on full width pages. Masi Typungato is a great example of how it is utilised.

JS, HTML and CSS are kept separate as it is easier to manage and to adjust code.

Control Flow and Loops

Control flow is a sequence of code that is executed based on a set of conditions. They use if / else statements to control the flow of code that is executed. If I was making a carrot cake for work colleagues and one of them was allergic to walnuts I would have to leave them out. The first part of the code asks whether someone is allergic to walnuts and if it’s true then I would use the recipe that excludes the walnuts. But if no one was allergic then I would bakeCake(); using walnuts. 

  
  if (allergicWalnuts === true) {
    bakeCakeWithoutWalnuts();
  }
  else {
    bakeCake(); 
  }

After baking the cake we would have to clean every baking utensil, appliance, cutlery and crockery. We can create a loop that repeats the process of stacking the dishwasher until the very last utensil is added OR the dishwasher is full. A loop must have an exit otherwise we‘ll create an infinite loop.

An array is a special variable that can hold more than one value at a time. 

  
    var devCohort = [“Roa”, “Tieke”, “Kauri”, “Kakapo”];

To access the values we refer to the index number. Say we wanted to sign our cohort “Tieke” to our statement would be: 

  
    var name = devCohort[1];

Objects are variables that can contain many values. Values are written as name:value where name is considered a property. In this example firstName, lastName, fluentMaori, and age are all properties. 

  
    var student = {
      firstName: “Pokare”, 
      lastName: ”Kareana”, 
      fluentMaori: true, 
      age: 44
    }

To access the object literal we use objectName.propertyName or objectName["propertyName"]. 

  
  student.lastName // accesses Pokare

So the difference is that to access a value from an array we use indexing and to access values from object literals we use dot notation or bracket notation.

Functions

A function is a block of code designed to perform a particular task. The code is executed when something calls it. The function syntax is outlined below: 

  
    function name(parameter1, parameter2) {
      code to be executed
    }

To call a function we say name(parameter1, parameter2) then the code within the curly brackets is executed. The code can include formulas, conversions, strings etc…

It’s reusable code that can be called anytime in your programme or site by code external to your function.
